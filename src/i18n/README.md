# i18n — Интернационализация

Система переключения языка интерфейса (RU/EN) с возможностью добавления новых.

## Файлы

| Файл | Назначение |
|---|---|
| `index.ts` | Движок: `useLocaleStore` (Zustand + persist в localStorage), `t(key, params?)`, `useT()` |
| `locales/ru.ts` | Словарь русских строк (~200 ключей, все существующие тексты интерфейса) |
| `locales/en.ts` | Английские переводы тех же ключей |

## Как работает

1. `useLocaleStore` хранит выбранную локаль (`'ru'`/`'en'`) и сохраняет в `localStorage`
2. Селектор в **Settings → Язык/Language** вызывает `setLocale()`
3. `t('key')` достаёт строку из активного словаря, при отсутствии — из `ru` (fallback), при отсутствии — возвращает сам ключ
4. `t('key', { count: 5 })` подставляет `{count}` в строку

## Как добавить новый язык

Создать `locales/xx.ts`, импортировать в `index.ts`, добавить в `LOCALES` и селектор в `SettingsDialog`.

## Зависимости

- `zustand` (только persist middleware)
