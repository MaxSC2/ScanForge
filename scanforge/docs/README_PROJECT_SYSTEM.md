# ⚙️ Основной процесс работы

## 1. Перед началом работы

* открыть `scanforge/docs/CURRENT_STAGE.md`
* понять цель этапа

---

## 2. Работа с задачами

* использовать шаблоны из /templates
* не выходить за scope этапа

---

## 3. Работа с ИИ

* давать контекст
* ограничивать scope
* не давать полный доступ ко всему проекту

---

## 4. Перед merge

* проверить pipeline
* пройти тесты
* убедиться в стабильности

---

## 5. Завершение этапа

* пройти `scanforge/docs/STAGE_GATE.md`
* убедиться, что всё стабильно

---

# 🚫 Запрещено

* начинать новые фичи вне этапа
* менять архитектуру без записи в `scanforge/docs/DECISIONS.md`
* игнорировать тестирование
* работать без `scanforge/docs/CURRENT_STAGE.md`
