# Vision — ScanForge

## What is ScanForge

ScanForge is a local-first desktop tool for scanlation workflows.

It provides:
- structured project storage
- region-based editing
- OCR + translation pipeline
- controlled rendering/export

The system is designed as a pipeline, not a drawing app.

---

## Who it is for

- solo scanlators
- small teams replacing manual Photoshop workflows
- developers building automated scanlation pipelines

---

## Core value

- replace fragmented tools with a single controlled system
- make scanlation reproducible and structured
- reduce manual repetitive work
- keep full local ownership of data

---

## What ScanForge is NOT (right now)

- not a cloud platform
- not a collaboration tool
- not a Photoshop replacement
- not an AI-only auto-translator
- not a marketplace or content platform

---

## North Star (next stages)

### Stage 5
- stable pipeline orchestration
- provider switching (OCR/translation)
- predictable export results

### Stage 6
- partial automation (batch jobs, queue system)
- improved UX for large projects

### Stage 7+
- optional integrations (external APIs)
- advanced pipeline control (automation hooks)

---

## Summary

ScanForge aims to become:
a reliable, local-first scanlation pipeline system
with controlled automation and predictable output
